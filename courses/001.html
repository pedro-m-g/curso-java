¿Qué es Java?

Java es un ecosistema de utilidades para realizar las siguientes actividades:

  1. edit Escribir programas en un lenguaje de programación estándar.
  2. construction Compilar el programa para generar código ejecutable.
  3. play_arrow Ejecutar el código compilado previamente.

El lenguaje de programación Java es un formato estándar de texto que puede ser interpretado por un compilador y posteriormente ejecutado. Este lenguaje define los bloques de construcción que se pueden utilizar en los programas y las reglas que éstos tienen que cumplir para ser válidos. Al conjunto de estas reglas se le llama sintaxis. Para que un programa pueda ser compilado, debe ser escrito con una sintaxis correcta.

El compilador tiene como función validar que los programas tengan una sintaxis correcta y, de ser así, convertir el texto a un formato ejecutable llanado bytecode. En caso contrario, el compilador mostrará los errores correspondientes, indicando el archivo, la línea de código y la razón del error ocurrido.

Finalmente, el código es ejecutado por la máquina virtual de Java, siguiendo las instrucciones descritas en el programa.

Instalación

El JDK (Java Development Kit) contiene todas las herramientas necesarias para desarrollar programas Java. Puedes descargar el JDK para tu sistema operativo desde el siguiente enlace:

https://adoptium.net/es/temurin/releases/?version=17

He elegido la versión 17 del JDK porque a la fecha es la versión LTS (Long Term Support) más reciente. Estas versiones son especiales porque reciben mantenimiento durante más tiempo, por lo que suelen ser más estables que el resto.

Para instalar Java, puedes descargar el archivo con extensión .msi (Instalador automático) o .zip (Archivos de instalación manual). A continuación se muestra el método de instalación manual (con el archivo zip), para que puedas verificar cada paso en caso de que la instalación automática no funcione correctamente.

  1. file_download Descargar el archivo zip del sitio de Adoptium.
  2. folder_zip Extraer el archivo zip en una ubicación conocida, como en "C:\Program Files".

  8. terminal Abrir la terminal cmd y (Puedes encontrar el programa "cmd" buscando en el menú de inicio de Windows) ejecuta el siguiente comando: 
    java -version
